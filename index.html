Quick Start

Resolve any DID using the Gatekeeper API:

# Resolve a did:cid identifier
curl https://archon.technology/api/v1/did/did:cid:bagaaieraxdxq4fm...

# Check node version
curl https://archon.technology/api/v1/version

# Use the Universal Resolver for any DID method
curl https://resolver.archon.technology/1.0/identifiers/did:web:example.com

Run Your Own Node

Archon is fully decentralized. Run your own Gatekeeper node to strengthen the network.

# Install the Archon CLI
npm install -g @didcid/archon

# Initialize and start your node
archon init
archon start

# Your node is now part of the network!
